# MiniProject เว็ปไซต์จองสนามกีฬา
    เว็บสำหรับอำนวยความสะดวกในการจองสนามกีฬา ช่วยให้ผู้ใช้จองสนามได้ทันที ส่วนแอดมินสามารถจัดการข้อมูลสนามและรายการจองได้อย่างครบวงจร
# ฟีเจอร์หลัก
    ระบบสมาชิก: User (ผู้จอง) และ Admin (ผู้ดูแล)
    การจัดการสนาม: แอดมินสามารถ เพิ่ม, แก้ไข และลบข้อมูลสนาม
    ระบบจองสนาม: ผู้ใช้สามารถเลือกวันที่ผ่านปฏิทิน และเลือกช่วงเวลา โดยระบบจะคำนวณจำนวนชั่วโมงให้อัตโนมัติ
    ประวัติการจอง: สามารถตรวจสอบรายการจองของตนเอง และยกเลิกรายการจองได้
